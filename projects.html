1) MiniP(i)C(i)
MiniP(i)C(i) also known as MNPC is a compact gaming console. It uses a Raspberry PI that is connected trough the GPIO pins to a breadboard with buttons that acts like a controller. It also has a LCD display where you can see the games we made using python and pygame. The purpose of this console is to cure boredoom while being a portable device.
